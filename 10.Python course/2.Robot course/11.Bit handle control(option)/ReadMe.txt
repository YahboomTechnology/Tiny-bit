1. Download the program to micro:bit board of Tiny:bit car. You need to use the MU software to open the tinybit-car-code.py file, and then click "Flash" to download.

2. Download the program to microbit board of the handle. You need to use the MU software to open the ghandle-code.py file, and then click "Flash" to download.

3. You must need to import the tinybit library file into the microbit first. If you don't know how to import it, please check [preparation before class]---> [Python programming]

4. After downloading the program, open the power switch on the handle and the car normally, they will automatically pair and can be controlled within the range of signal transmission.
